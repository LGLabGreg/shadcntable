## Prerequisites

This project assumes you have shadcn/ui installed in your project. If you haven't, follow the instructions on the shadcn website: <a className='underline' href='https://ui.shadcn.com/docs/installation' target='_blank'>https://ui.shadcn.com/docs/installation</a>

## Install shadcntable

The following command will install the shadcntable components and utils in `/components/shadcntable`

```bash
pnpm dlx shadcn@latest add https://shadcntable.com/r/data-table.json
```
